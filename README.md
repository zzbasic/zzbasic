# ZzBasic
ZzBasic é uma linguagem de programação que combina a simplicidade icônica e a sintaxe legível do BASIC com recursos expressivos inspirados em Python e o poder de baixo nível de C/C++. Criada para ser acessível a iniciantes, ela oferece um caminho suave para aprender conceitos fundamentais sem abrir mão da capacidade de construir projetos robustos.
